# OpenVINO Face Detection

Real-time face detection using Intel OpenVINO toolkit.

## What it does
- Loads Intel's face-detection-0200 model
- Runs real-time inference on webcam feed
- Draws bounding boxes with confidence scores
- Runs on CPU using OpenVINO runtime

## Setup
pip3 install openvino opencv-python numpy

## Run
python3 face_detect.py
